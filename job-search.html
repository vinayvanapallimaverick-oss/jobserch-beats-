RapidAPI Setup

Provider: RapidAPI JSearch only.
Security: API key stays on the Node server.
Search: role, location, days, job type, remote, and resume skills.
Speed: live API calls depend on RapidAPI network latency.
Backend flow
  1. 1. BrowserSends search filters and detected resume skills to the local backend.
  2. 2. Node serverCalls RapidAPI JSearch with your private key from the environment.
  3. 3. NormalizerConverts JSearch fields into clean cards and local match scores.
Local setup
Keep .env private. If you change the RapidAPI key, restart the Node server so it can reload the environment.
